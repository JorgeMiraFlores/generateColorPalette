# Color Palette Generator

A very simple color palette generator. 

## Usage:

1. Write the amount of colors you want to generate.
2. Select the RGB index sort:
   * Red RGB index:
       - from highest to lowest.
       - from lowest to highest.
   * Green RGB index:
       - from highest to lowest.
       - from lowest to highest.
   * Blue RGB index:
        - from highest to lowest.
       - from lowest to highest.
